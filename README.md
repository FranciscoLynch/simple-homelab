## 1. Install

- Vagrant: https://www.vagrantup.com/docs/installation
- Oracle Virtualbox: https://www.vagrantup.com/docs/providers/virtualbox

### Clone the repository

```
git clone https://github.com/FranciscoLynch/simple-homelab.git
```

### Enter the lab

```
cd simple-homelab 
vagrant up
```

Once it's done

```
vagrant ssh
```
